# Process Safety Support Website

Explore emerging Process Safety and Loss Prevention consultancy solutions for Oil & Gas, Process Industries, and Insurance Risk Management.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

If you want to work locally, ensure you have Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Deployment

Deploy the `dist/` build folder to any static hosting provider (like Vercel, Netlify, or GitHub Pages) by running:

```sh
npm run build
```
